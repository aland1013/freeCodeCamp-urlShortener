API Project: URL Shortener Microservice

User Story:

  1. I can POST a URL to https://freecodecamp-project-urlshortener-abl.glitch.me/api/shorturl/new and I will receive a shortened URL in the JSON response.
    Example : {"original_url":"www.google.com","short_url":"83030"}
  2. If I pass an invalid URL that doesn't follow the https://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"}
  3. When I visit the shortened URL, it will redirect me to my original link.
    Example:
    https://freecodecamp-project-urlshortener-abl.glitch.me/api/shorturl/83030
    redirects to
    https://www.google.com
